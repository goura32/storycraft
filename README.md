# Storycraft

LLMと一緒に、長編の物語を**最後まで書き切る**ための道具。

## 目指すこと

- 利用者が、題材・登場人物・雰囲気・結末を決める。
- 道具が、場面ごとの計画と本文作成を支援する。
- 利用者は重要な分岐で選び直せる。
- 途中で設定を増やしすぎず、全3巻程度の物語を完結させる。
- 最終的に読みやすいMarkdownまたはEPUBとして出力する。

## 最初に作らないもの

- 完全自動で品質を保証する判定機構
- 複雑な内部状態の厳密照合
- 複数の保存場所に同じ設定を複写する仕組み
- 共同執筆、公開サイト、課金、画像生成

## 今の段階

要件定義中。実装は、最初の利用場面と最小の画面・データを決めてから開始する。
